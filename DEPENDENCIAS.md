# Dependencias

**Desarrollo:** Node, npm, Git, Android Studio (emulador), PostgreSQL + pgAdmin, Expo Go. Backend dev: nodemon. Frontend dev: TypeScript, ESLint.

**Producción backend:** cors, dotenv, express, google-auth-library, pg. Instalar con `npm install` en `backend/` (o `npm install --production` para solo dependencies).

**Frontend (build):** expo, react, react-native, expo-router, expo-auth-session, expo-crypto, @react-native-async-storage/async-storage, y el resto del package.json. No se “instalan” en servidor; se empaquetan en el build (Expo EAS, web, etc.).
